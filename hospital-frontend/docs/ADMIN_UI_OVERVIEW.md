# 管理端全新界面说明

- **入口与标签页**：访问 `/admin` 默认打开“仪表盘”标签，侧栏点击会自动新建/切换对应标签，除仪表盘外均可关闭。
- **顶栏**：展示当前账户名称与角色，右侧提供“退出登录”按钮（前端模拟跳转首页）。
- **侧栏导航**：仪表盘、用户管理、科室管理、排班管理、挂号管理、统计报表、公告管理。文本与图标写死在前端，便于后续对接接口。
- **功能模块（均使用模拟数据展示）**：
  - 仪表盘：今日排班/挂号速览、实时动态、科室负载、公告播报。
  - 用户管理：角色/状态/关键词筛选，表格列出管理员、医生、患者的联系与状态信息。
  - 科室管理：卡片+表格展示负责人、医生数、诊室数及当前重点与状态。
  - 排班管理：按日期/科室/状态筛选医生档期，展示号源占用与状态。
  - 挂号管理：状态、科室、关键词筛选；挂号列表包含渠道、类型、备注。
  - 统计报表：6 个月挂号与收入趋势模拟柱状展示，医生表现榜。
  - 公告管理：公告状态（草稿/预告/已发布）、受众范围与负责人列表。
- **样式**：全新布局位于 `src/components/admin/AdminLayout.tsx` 与 `AdminLayout.css`，页面组件在 `src/pages/admin/` 下按功能拆分。
- **构建检查**：在 `hospital-frontend` 目录执行 `npm run build` 可验证通过（使用本地依赖，无后端请求）。
